# pkgui
Manage arch linux packages easily (including aur)

## what it is
This is a tiny tui frontend for installing and removing packages in arch linux and arch based distros.

## How to use it
Using this is pretty straightforward, run pkgui, then you can see 3 menues. Pac install(for installing official packages), yay install(for install aur packages), remove package to remove the package.

## Prerequisites
You should have :
fzf
yay
gum
bash
